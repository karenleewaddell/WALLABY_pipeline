Docker images

We use official AusSRC docker images for executing the Nextflow pipelines. The default images can be updated if necessary. The Nextflow parameters for these images are given below.

ParameterDefault ImageDescription
WALLABY_COMPONENTS_IMAGE_IMAGEaussrc/WALLABY_COMPONENTS_IMAGE:latestCollection of Python scripts for WALLABY workflow components. This includes the generation of configuration files, downloading image cubes and checksum calculations. Repository found here.
S2P_IMAGEaussrc/s2p_setup:latestPython script for the generation of sofia.par and config.ini required to run sofia and sofiax. Adapted from SoFiA-Admin repository found here for Nextflow execution.
SOFIA_IMAGEDocker image for sofia execution.astroaustin/sofiax:v0.0.5*
SOFIAX_IMAGEDocker image for sofiax execution.aussrc/sofiax:latest

*Official image pending.
